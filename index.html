ANÁLISE DE REGISTROS DA FAUNA MARINHA NO BRASIL

Para apoiar a gestão da conservação marinha, este projeto analisa registros do Projeto de Monitoramento de Praias (Petrobrás e IBAMA) de 2020 a 2024. Os dados foram tratados em Python para limpeza, padronização e análise exploratória, e os resultados foram organizados em um dashboard no Power BI. A análise destaca a distribuição geográfica dos registros, a presença de espécies ameaçadas, os padrões de ocorrência entre grupos taxonômicos e as variações sazonais, permitindo identificar áreas sub-representadas e otimizar estratégias de monitoramento e conservação.

Ferramentas:

  • Análise e manipulação de dados: Python (Jupyter Notebook, Pandas)
  • Visualização e dashboards: Microsoft Power BI
  • Gestão de dados: Google Sheets, arquivos CSV
  • Apresentação e design: Canva
  • Controle de versão: Git e GitHub

CLASSIFICAÇÃO DE DOENÇA CARDÍACA POR MODELOS DE MACHINE LEARNING

Com objetivo de comparar a performance dos modelos de machine learning K-Nearest Neighbors (KNN) e Random Forest para prever o diagnóstico de doença cardíaca em pacientes, utilizei dados com informações sobre a saúde de pacientes. Os modelos foram avaliados com base na sua capacidade de classificar corretamente aqueles com e sem a doença. Os resultados mostraram que o modelo Random Forest teve um desempenho superior, sendo mais eficaz na identificação dos pacientes com risco de doença cardíaca.

Ferramentas:

  • Análise e manipulação de dados: Python (Pandas, Numpy)
  • Visualização de dados: Matplotlib
  • Modelagem e aprendizado de máquina: Scikit-Learn
